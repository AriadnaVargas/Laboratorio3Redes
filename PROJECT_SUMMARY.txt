🔧 SOCKET FUNCTIONS USED:

CORE:
  socket()    - Create socket
  bind()      - Bind to port (server only)
  listen()    - Listen for connections (server only)
  accept()    - Accept connections (server only)
  connect()   - Connect to server (client only)
  send()      - Send data
  recv()      - Receive data
  close()     - Close socket

UTILITY:
  htons()     - Host to network byte order
  ntohs()     - Network to host byte order
  inet_aton() - Convert IP string to binary
  inet_ntoa() - Convert binary IP to string

STRING:
  memset()    - Initialize memory
  strcpy()    - Copy strings
  strlen()    - String length
  strcmp()    - Compare strings
  strtok()    - Tokenize strings

SYSTEM:
  sleep()     - Pause execution
  signal()    - Handle signals
  strerror()  - Error messages
